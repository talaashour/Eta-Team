# Interactive Command Tutor# Interactive Command Tutor

## Project Description

This project is an interactive command-line quiz application built using Python.

It helps students practice and review Linux shell commands taught in the Systems Programming course.

## Features

- 20 multiple-choice questions

- Immediate feedback for correct and wrong answers

- Input validation for a, b, c, and d only

- Final score displayed at the end

- Colored terminal output for correct and incorrect answers

## Technologies Used

- Python

- Lists and dictionaries

- For loops and while loops

- If/else statements

- String methods

- ANSI escape codes

## How to Run

1. Download or clone the repository.

2. Open the terminal in the project folder.

3. Run the program using:

```bash
python interactive\_command\_tutor.py
